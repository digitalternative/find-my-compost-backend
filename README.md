## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
# install nest cli globally
npm i -g @nestjs/cli
# create project
nest new find-my-compost-backend
# install dependencies and graphql
npm i @nestjs/graphql graphql@^15 apollo-server-express apollo-server-core --save
# Add a ressource no controller needed for graphql (only one endpoint)
nest g resource composts
# ODT (MongoDB) integration
# Install a local mongoDB instance: https://docs.mongodb.com/manual/administration/install-community/
#https://docs.nestjs.com/techniques/mongodb
npm install --save mongoose@nestjs/mongoose  mongoose
# ORM integration
# https://docs.nestjs.com/techniques/database
# see typeOrm: https://orkhan.gitbook.io/typeorm
npm install --save @nestjs/typeorm typeorm mysql2
```

 ## GraphQl queries examples

create compost
 ```bash
 
  mutation {createCompost(
    createCompostInput: [{
      title: "Composte de Chastre",
      email: "chastre-composte@mail.be",
      type: "outdoor",
      phone: "010 99 99 99",
      website: "chastre.be",
      address: {
        street: "Rue du piroy",
        city: "Chastre",
        zipcode: 1450,
        coordinates: {
          lat: 50.60450408661511,
          lng: 4.635640580663463,
      	}
      }
    }]
  ){
    title
  }
}

 ```
Display composts
 ```bash
 query {composts{_id,title,email,address{street,city,zipcode,coordinates{lat,lng}},phone,website,type}}
  ```
Display compost
 ```bash
  query {compost(_id: "61e6a7f57395d53f924e2ea2"){_id,title,email,address{street,city,zipcode,coordinates{lat,lng}},phone,website,type}}
```

Update compost
 ```bash
 mutation {updateCompost(
     updateCompostInput: {
      _id: "61e6a7f57395d53f924e2ea2"
      title: "Composte de Tombouktou"
    }) {
     _id
 	}
 }
 ```
 Remove compost
  ```bash
 mutation {removeCompost(_id: "61e6a7f57395d53f924e2ea2"){_id, title,email,address{street},phone,website,type}}
  ```
